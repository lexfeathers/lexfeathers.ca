I've had the pleasure of handling sound design and direct Unity implementation with FMOD. In addition, this was my first time working as a junior gameplay programmer, which has been a wonderful learning experience in my journey to shore up my programming fundamentals and get more creative with the interactive aspects of game audio.

For the programming part of my role, I have been responsible for features such as the Day/Night cycle, a modular audio system with FMOD integration, and many, many bug fixes, all in C# using MonoBehaviour scripts and Scriptable Objects. 

For the sound design part, I've been in charge of creating bespoke sounds for the world, characters, items, and UI. Building the technical systems really gave me creative freedom with how I could implement the audio, so I had a blast adding details I wouldn't normally be able to in most games without dedicated help from the dev team.

TaleKeep is currently in early development with a release date TBA.
